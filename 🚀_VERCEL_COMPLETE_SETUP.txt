========================================
 🔍 TROUBLESHOOTING
========================================

Build Fails:
------------
- Check build logs in Vercel
- Verify all 8 env vars are added
- Check for TypeScript errors
- Redeploy after fixing

Mixed Content Error:
-------------------
- Verify proxy route uses HTTPS
- Check: app/api/proxy/route.ts
- Should be: https://api.recapca.com
- NOT: http://13.203.232.46:8000

401 Unauthorized:
----------------
- Check Firebase credentials
- Verify all 7 Firebase env vars
- Test login/signup flow
- Check browser console

Payment Not Working:
-------------------
- Verify Razorpay key is added
- Check: NEXT_PUBLIC_RAZORPAY_KEY_ID
- Test payment modal opens
- Check browser console

API Calls Failing:
-----------------
- Verify backend is running
- Test: https://api.recapca.com/health
- Check proxy route is correct
- Verify all endpoints use /api/proxy/api/
